# Chemistry Element Study Tool

A command-line interface (CLI) tool designed to help you study and compare elements from the periodic table using a JSON-based database.

## Features
- **Study Element:** Look up detailed properties of any element by name, symbol, or atomic number.
- **Compare Elements:** View side-by-side comparisons of two elements.
- **Filter by Period:** List all elements belonging to a specific period.
- **Random Study:** Get a random element to test your knowledge.

## Setup
1. Clone this repository to your local machine.
2. Ensure you have the data file located at `data/elements.json`.
3. Run the main menu:
   ```bash
   python3 main.py
